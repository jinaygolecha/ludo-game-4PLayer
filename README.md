## Project Description
This project is a **4-player Ludo game built using Python**.  
The goal of this project was to recreate the classic Ludo board game using programming concepts such as game logic, player turns, dice mechanics, and collision detection.

The game includes a **computer bot player** and supports multiple players with turn-based gameplay.

## Features
- 4 player Ludo gameplay
- Dice rolling mechanics
- Player turn management
- Piece movement logic
- Collision and kill rules
- Basic graphical interface using Python

## Technologies Used
- Python
- Game logic programming
- Object-Oriented Programming

## How to Run the Game

1. Clone the repository


git clone https://github.com/jinaygolecha/ludo-game.git


2. Navigate to the project


cd ludo-game


3. Install dependencies


pip install -r requirements.txt


4. Run the game


python main.py
